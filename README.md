Android-Utils
========

Combination of common Android Utils

Download
--------

Gradle:

Add the JitPack repository to your project level gralde file

```groovy
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
  
Add the dependency

```groovy
compile 'com.github.duyp:android-utils:1.0.6'
```
